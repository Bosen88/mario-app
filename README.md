# 超級瑪利歐破關大作戰 🍄

嘉南執行團隊 2026 年 5–6 月招募競賽 Web App

## 功能
- 登入系統（首次設定姓名、角色、密碼）
- 任務上傳（含截圖證明）
- 即時計分看板（人均戰力固定除以 27）
- 個人排行榜前 10 名
- 管理者後台
- 遊戲音效

## 帳號
- 一般成員：選擇小隊名稱，首次登入自行設定密碼
- 管理者：帳號「大C」，密碼 `admin888`

## 本地開發
```bash
npm install
npm run dev
```

## 部署
直接 push 到 GitHub，連接 Vercel 自動部署。

## 資料儲存
使用 `localStorage`，資料存在瀏覽器本地，清除瀏覽器快取會重置資料。
